README file for R package and data supporting the paper "Differential network analysis from cross-platform gene expression data".


Contents of this archive
------------------------
This archive contains several R package and ovarian cancer gene expression data used in the above paper. 

(1) TDJGL_1.0.tar: R package .
(2) OV_GeneExp.Rdata: Gene expression data. Before using this data, we suggest normalizing the data to have mean 0 and 
standard deviation 1 within each patient group.
